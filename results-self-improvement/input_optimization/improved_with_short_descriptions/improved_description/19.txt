The user initiates the process by logging in. Following successful login, the user selects desired items.  Subsequently, the user selects a payment method.  There is an exclusive choice between selecting a reward and proceeding without a reward.  If the user chooses a reward, the reward is applied to the order. The user then proceeds to the payment stage.  There is an exclusive choice between completing payment in full and entering into an installment agreement. Upon successful payment or agreement completion, the order is processed for delivery, and the user receives the delivery. Following delivery, the user has the option to return items. For each returned item, a new delivery of the exchanged item is initiated.  Each return initiates a separate delivery process.
